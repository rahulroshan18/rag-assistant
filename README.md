# Resume RAG Assistant

An AI-powered Retrieval-Augmented Generation (RAG) assistant built using:

- Python
- OpenAI Embeddings
- ChromaDB
- OpenAI GPT-5
- PDF Processing (PyPDF)

## Features

- PDF ingestion
- Intelligent text chunking
- Semantic embeddings
- Vector search using ChromaDB
- Natural language querying

## Architecture

PDF
→ Chunking
→ Embeddings
→ ChromaDB
→ Semantic Search
→ GPT Answer
